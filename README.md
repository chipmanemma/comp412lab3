To run:
- type "make clean" into the console to clean any .class files
- type "make" or "make build" to remake the class files 
- run the main program by using the command "./schedule <flags> <file>"
- If you want to see the graphs, uncomment lines starting with graph.append and the string builder as well as the call to the function viewGraph(graph);
    - you will need to change the file path in viewGraph to somewhere on your computer
